# USDX Whitepaper

## 1. Introduction
USDX is a BEP-20 utility token deployed on the BNB Smart Chain.

## 2. Problem Statement
Many blockchain users require simple and transparent tokens for testing,
education, and peer-to-peer transfers without complex mechanics.

## 3. Solution
USDX provides a lightweight BEP-20 token designed for transparency,
simplicity, and experimental usage within the BSC ecosystem.
