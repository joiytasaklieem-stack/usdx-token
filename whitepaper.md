# USDX Whitepaper

## Overview
USDX is a BEP-20 utility token deployed on the BNB Smart Chain.

The project is designed for peer-to-peer transfers, testing,
and experimental on-chain interactions with decentralized
applications within the BSC ecosystem.

## Token Information
- Name: USDX
- Network: BNB Smart Chain
- Standard: BEP-20
- Contract Address:
0x01E4a7Ba918a1317218E562581c51bDfF08b7cE7

## Token Distribution
USDX did not conduct any ICO, IEO, private sale, or seed sale.
All tokens were created at deployment.

## Purpose
USDX is intended for utility and experimental use only.
The project does not provide investment guarantees.
